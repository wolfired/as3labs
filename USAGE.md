# 用法

## 用户目录下新建 `mm.cfg`

```
TraceOutputBuffered=1
TraceOutputFileEnable=1
ErrorReportingEnable=1
AS3Verbose=0
AS3Trace=0
```

## 日志文件 `flashlog.txt`

只有使用调试版Flashplayer才会生成

## 新建 `./bin/debug/cfg`

```
preloads=demo_core.swf,demo_module_entry.swf,demo_module_terminal.swf
```

## 放一张 `./bin/debug/loadingbg.jpg`
